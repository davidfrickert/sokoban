# sokoban
Simple sokoban game written in Rust


# Installation

Requires `git` 

Clone repository
```
git clone https://github.com/davidfrickert/sokoban.git
```
# Running

Requires `Cargo` 

Go into cloned folder first (if you are in the folder you called `clone`)
```
cd sokoban
```
Then run
```
cargo run --release
```
